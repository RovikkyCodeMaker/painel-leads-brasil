# 📊 Painel Leads Brasil (LeadMiner Pro B2B) 🚀

Uma plataforma web inteligente de mineração, centralização e tratamento de dados comerciais de alta performance. O sistema foi projetado para mapear, filtrar e estruturar leads corporativos de Norte a Sul do Brasil, oferecendo uma interface analítica escura (Dark/Clean) e exportação automatizada para inteligência de mercado.

Este projeto resolve uma dor real de empresas de vendas (B2B): a perda de tempo com a busca manual de clientes e a desorganização de contatos comerciais.

---

## 🎯 Principais Funcionalidades

* **Arquitetura API First:** Construído com FastAPI para garantir respostas ultra-rápidas e baixíssimo consumo de hardware (memória RAM e processamento).
* **Persistência em Banco de Dados:** Integração com SQLite3 para armazenar leads de forma segura, estruturada e protegida contra perda de dados.
* **Filtro Territorial Dinâmico:** Interface assíncrona baseada em JavaScript (Fetch API) e Tailwind CSS que permite filtrar empresas por estados em tempo real, sem recarregar a página.
* **Engenharia de Dados (Exportação Pro):** Motor integrado com Pandas e OpenPyXL para converter dados brutos do banco de dados em planilhas Excel (`.xlsx`) perfeitamente formatadas com apenas um clique.
* **Segurança Avançada:** Consultas SQL parametrizadas para proteção total contra ataques de *SQL Injection*.

---

## 🛠️ Tecnologias Utilizadas

O ecossistema do projeto foi desenvolvido utilizando tecnologias modernas e altamente requisitadas no mercado de back-end e automação:

* **Back-end:** Python 3, FastAPI, SQLite3
* **Front-end:** HTML5, Tailwind CSS (Design Responsivo/Dark), JavaScript Assíncrono
* **Manipulação de Dados:** Pandas, OpenPyXL
* **Servidor de Aplicação:** Uvicorn
